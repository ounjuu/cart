## :hammer: 주요기능
### [ 관리자 페이지 ]
#### <관리자 페이지에서 상품 등록>
1. 상품 등록 시 oninput으로 조건 검사 후 저장 버튼 활성화
2. 상품 등록을 하면 이미지 랜덤으로 나옴
3. 하단의 상품은 수정 및 삭제 가능(로컬스토리지도 수정 및 삭제)<br/>
